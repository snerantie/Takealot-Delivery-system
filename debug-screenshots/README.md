# Debug screenshots

Upload your Render and Vercel error screenshots into this folder so they can be reviewed.

How to upload on GitHub:
1. Open this `debug-screenshots` folder on GitHub.
2. Click **Add file → Upload files**.
3. Drag your screenshots in, then **Commit changes**.

Suggested names so it's clear which is which:
- `render-build-error.png`
- `render-logs.png`
- `vercel-build-error.png`
